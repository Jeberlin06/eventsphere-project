# EventSphere – Event Booking Web Application

## 🔗 Live Demo
https://eventsphere-event.netlify.app/
## 📌 Features
- Event filtering and search
- Ticket booking system
- Dynamic price calculation
- PDF receipt generation
- Admin dashboard

## 🛠 Technologies Used
HTML, CSS, JavaScript, Bootstrap, Netlify

## 📖 Description
EventSphere is a web-based application that allows users to browse and book events easily.
